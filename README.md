A discord bot made with [discord.py](https://github.com/Rapptz/discord.py).

> Setup

```shell
git clone https://github.com/SyedAhkam/jeju-bot.git
cd jeju-bot
pipenv shell
touch .env
nano .env[place your token in there and mongodb srv]
pip3 install dependencies
python3 bot.py
```
